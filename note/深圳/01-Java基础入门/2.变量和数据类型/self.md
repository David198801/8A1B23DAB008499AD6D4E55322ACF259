单目运算符：运算所需变量为一个的运算符

| 优先级 | 运算符 | 结合性 |
| - | - | - |
| 1 | ()、[]、{} | 从左向右 |
| 2 | !、+(正)、-(负)、~、++、-- | 从右向左 |
| 3 | \*、/、% | 从左向右 |
| 4 | +、- | 从左向右 |
| 5 | «、»、&gt;&gt;&gt; | 从左向右 |
| 6 | &lt;、&lt;=、&gt;、&gt;=、instanceof | 从左向右 |
| 7 | ==、!= | 从左向右 |
| 8 | &amp; | 从左向右 |
| 9 | ^ | 从左向右 |
| 10 | | | 从左向右 |
| 11 | &amp;&amp; | 从左向右 |
| 12 | || | 从左向右 |
| 13 | ?: | 从右向左 |
| 14 | =、+=、-=、\*=、/=、&amp;=、|=、^=、~=、«=、»=、&gt;&gt;&gt;= | 从右向左 |
