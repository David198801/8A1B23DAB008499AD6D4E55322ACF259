### 所用工具

[HCADecoder](https://github.com/Nyagamon/HCADecoder)

### 步骤

推特上有大佬公开了密钥：0000000001395C51

因此首先运行以下命令进行解密：

```
.\hca.exe -c -a 01395C51 -b 00000000 HCA文件

BASH
```

之后再转换即可：

```
.\hca.exe HCA文件
```
